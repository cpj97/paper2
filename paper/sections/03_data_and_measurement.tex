\section{Data and Measurement}

\subsection{Unit of Analysis}

The unit of analysis is a grid cell by year panel. Explain:

\begin{itemize}
    \item grid resolution,
    \item time span,
    \item spatial coverage,
    \item the forest-loss outcome construction.
\end{itemize}

\subsection{Treatment Construction}

Define the treatment layers clearly:

\begin{itemize}
    \item protected areas from WDPA yearly snapshots,
    \item Afro titles,
    \item Indigenous titles,
    \item Campesino reserves,
    \item any combined land-title treatment if used.
\end{itemize}

Important details to document:

\begin{itemize}
    \item data source and raw file provenance,
    \item geometry cleaning choices,
    \item overlap rule used to mark a cell as treated,
    \item how first-treatment years are assigned,
    \item how pre-panel treatment is handled,
    \item whether treatment status is absorbing by construction.
\end{itemize}
